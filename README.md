# CNC Marsbot - Assembly Mips

## Demo
* Clone from Github  
```  
git clone https://github.com/HuyennoGitHub/postscript-cnc-marsbot  
```
* Install MARS4_5.jar
* Open Marsbot and Digital Lab Sim
* Run code, choose 0, 4 or 8 to see the result
